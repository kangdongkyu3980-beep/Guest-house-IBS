# references

## 목적

이 폴더는 외부 참고 자료 원본을 보관하는 곳이다.

원본 PDF나 문서는 여기 두되, 실제 작업 기준은 이 폴더에만 두지 않는다.  
참고 자료에서 취사선택한 결과는 반드시 저장소 문서로 다시 정리한다.

## 현재 참고 자료

- `Codex로_일하는_법_260407_174701.pdf`

## 이 저장소에 반영한 원칙

위 PDF를 참고해 아래 원칙을 채택했다.

1. `AGENTS.md`는 짧은 지속 규칙으로 유지한다.
2. 큰 설명과 작업 방식은 `docs/` 문서로 보낸다.
3. 큰 변경은 먼저 계획으로 자른다.
4. 작업 요청은 가능하면 이슈처럼 구조화한다.
5. 다음 세션이 문서만 읽고 다시 시작할 수 있게 현재 상태를 남긴다.
6. 실행 계획은 `docs/exec-plans/active/`에 유지한다.

## 적용하지 않거나 보류한 것

아래 항목은 지금 저장소 단계에서는 보류한다.

1. 대규모 병렬 작업 중심 운영
2. 멀티 PR 중심 워크플로
3. 환경 자동화에 대한 큰 투자

이유는 현재 우선순위가 운영 안정화, 문서화, 구조 경계 정리이기 때문이다.
